Ssh免密登录配置

写于 2020-01-01 · 分类： Linux

本地机器配置免密码登录远程linux服务器

ssh-keygen #生成公钥和私钥
ssh-copy-id -i ~/.ssh/id_rsa.pub root@目标IP #将公钥追加到部署目标服务器
ssh root@目标IP # 测试免密登录
